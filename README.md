# pcmanfm-qt

pcmanfm-qt is the Qt port of the PCManFM file manager, designed to be the default file manager for the LXQt desktop environment.

## Features
- **Lightweight & Fast:** Designed for performance and low memory footprint.
- **Tabbed Browsing:** Manage multiple folders in a single window.
- **Advanced Interface:** Clean, customizable UI with drag-and-drop support.
- **Desktop Management:** Handles desktop icons and wallpapers.
- **Integration:** Seamless integration with the LXQt desktop environment.

## Installation and Usage

### Installation
pcmanfm-qt is available in the official repositories of most Linux distributions.

**Debian/Ubuntu:**
```bash
sudo apt install pcmanfm-qt
```

**Fedora:**
```bash
sudo dnf install pcmanfm-qt
```

**Arch Linux:**
```bash
sudo pacman -S pcmanfm-qt
```

### Usage
To launch the file manager, run the following command in your terminal or select "PCManFM-Qt" from your application menu:

```bash
pcmanfm-qt
```
